# Architecture Decision Record 003: PDF Generation Using Pug Templates and html-pdf

## Status
Accepted

## Context

The system requires generating PDFs for invoices, quotes, offers, and payments with customizable templates.

## Decision

Use Pug templating engine to create HTML templates for each document type. Use `html-pdf` to convert rendered HTML to PDF files saved on disk.

## Consequences

- Templates are easy to customize and maintain.
- PDF generation is integrated into backend workflows.
- PDFs are stored in organized folders for retrieval.
- Allows callback hooks after PDF generation for further processing (e.g., emailing).
